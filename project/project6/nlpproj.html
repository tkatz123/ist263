Fake News Detection Using NLP

In an age of misinformation, I built an NLP-based machine learning pipeline to classify news articles as real or fake with high accuracy. The project uses logistic regression, random forest, and a meta-classifier to combine model outputs and generate robust, confidence-scored predictions. It also features a Streamlit app that allows users to input articles for real-time analysis.

Dataset

Sourced from Kaggle
The dataset includes 72,134 labled articles (real/fake) aggregated by a team of researchers from mutliple sources including Reuters and BuzzFeed.

Methodology

  1. Exploratory Analysis — Visualized top keywords and sentiment differences between real and fake news
  2. Text Preprocessing — Cleaned and normalized text, removed noise, and engineered sentiment features
  3. Feature Engineering — Combined TF-IDF vectorized text with sentiment polarity scores
  4. Modeling:
    • Trained Logistic Regression and Random Forest classifiers
    • Developed a meta-classifier to combine their output probabilities using logistic regression
  5. Evaluation — Measured performance using accuracy, precision, recall, F1-score, and calibration plots
  6. Deployment — Built a Streamlit interface to test articles in real time and explore model predictions

Results

Highlights

  • Used swifter to dramatically speed up sentiment feature generation across 70k+ articles
  • Tuned model hyperparameters with GridSearchCV to reduce overfitting
  • Built an intuitive streamlit interface where users can:
    • Paste article text to get real/fake predictions + confidence scores
    • Upload their Kaggle API key to download and explore the dataset themselves
